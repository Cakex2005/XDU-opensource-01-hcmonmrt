## 个人介绍

👋 你好，我是王亦冕 ([KamikazEr101](https://github.com/KamikazEr101))！

一名来自西安电子科技大学的软件工程师，专注于用 AI 为软件开发和现实世界赋能。

- 🧠 **AI 应用工程师**：我热衷于将 AI Agent 和深度学习模型融入软件。
  - **[ClasSmart](https://github.com/KamikazEr101/ClasSmart)**：一个智能垃圾分类系统。它的 Python 后端不仅用 PyTorch 做图像识别，还用 Autogen Agent 防御恶意图片。
  - **[docker_compose_generator](https://github.com/KamikazEr101/docker_compose_generator)**：一个基于 CrewAI 的多智能体工具，能自动分析你的项目，一条龙生成 Dockerfile 和 docker-compose.yml。
  - **[GoodByeSqlBoy](https://github.com/KamikazEr101/GoodByeSqlBoy)**：一个让你用“人话”和数据库交流的工具，能将自然语言直接转换成 SQL 语句。
- 💻 **Java 后端开发者**：擅长用 Spring Boot 全家桶构建稳定可靠的后端服务。
  - **[ShangTing-Apartment](https://github.com/KamikazEr101/ShangTing-Apartment)**：一个功能完善的公寓租赁管理系统后端。
  - 在 **[ClasSmart](https://github.com/KamikazEr101/ClasSmart_backend_java)** 项目中，我也负责了处理主要业务逻辑的 Java 微服务，并用 Redis 解决了分布式部署中的一些有趣问题。
- 🚀 **探索者**：目前也在参与一个保密的多智能体系统 (MAS) 科研项目，持续探索 AI 的边界。
