## 🎯 团队目标 (Team Goals)

通过本课程，我们团队希望共同达成以下目标：

-   🌍 **深入理解开源**：探索开源项目背后的协作模式、社区规范和价值观。
-   🤝 **精通 Git & GitHub**：熟练掌握企业级的 Git 分支策略与团队协作流程。
-   🌱 **贡献开源社区**：迈出为开源项目贡献代码、文档或解决方案的第一步。
-   🛠️ **提升工程能力**：将理论知识应用于实践，锻炼解决真实问题的能力。

---

## 🔧 技术方向与兴趣 (Tech & Interests)

我们的团队对以下技术领域充满热情，并计划在课程项目中进行探索：

-   **人工智能赋能软件工程 (AI4SE)**：利用 AI 提升开发效率与软件质量。
-   **智能运维 (AIOps)**：探索 AI 在自动化系统监控与管理中的应用。
-   **多智能体系统 (MAS)**：研究多个智能体协作解决复杂问题的架构与实现。
-   **游戏开发 (Game Development)**：关注开源游戏引擎及相关工具链。
-   **前端与UI设计 (UI/UX)**：致力于构建用户体验优秀的设计系统与组件库。
